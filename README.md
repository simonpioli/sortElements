    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><a name="sort"></a><h2>sort</h2>

<p>The sort plugin is a very basic low-level element sorting function that allows you to sort DOM elements with a custom comparator (similar to <code>Array.prototype.sort</code>).</p>

<p>Assuming the following markup:</p>

<pre><code>&lt;ul&gt;
    &lt;li&gt;Banana&lt;/li&gt;
    &lt;li&gt;Carrot&lt;/li&gt;
    &lt;li&gt;Apple&lt;/li&gt;
&lt;/ul&gt;
</code></pre>

<p>You could sort the items alphabetically like so:</p>

<pre><code>$('li').sortElements(function(a, b){
    return $(a).text() &gt; $(b).text() ? 1 : -1;
});
</code></pre>

<p>That would result in:</p>

<pre><code>&lt;ul&gt;
    &lt;li&gt;Apple&lt;/li&gt;
    &lt;li&gt;Banana&lt;/li&gt;
    &lt;li&gt;Carrot&lt;/li&gt;
&lt;/ul&gt;
</code></pre>

<p>It also let's you specify what element will be sorted. The current collection's elements will be those referred to as <code>a</code> and <code>b</code> on each call of the comparator, but you might not want those elements to be the ones to move. E.g. you might want it to be a parent. For example, when sorting a table column, you would sort by the <code>&lt;td&gt;</code> elements, but the elements you actually want to move within the DOM are the <code>&lt;tr&gt;</code> (each <code>&lt;td&gt;</code>'s parent):</p>

<pre><code>$('td').sortElements(myComparator, function(){
    // Return a reference to the desired element:
    return this.parentNode;
});
</code></pre>

<p>See more info here: <a href="http://james.padolsey.com/javascript/sorting-elements-with-jquery/">http://james.padolsey.com/javascript/sorting-elements-with-jquery/</a>.</p></article>

  <h2>Markdown Cheat Sheet</h2>

  <div class="cheatsheet-content">

  <div class="mod">
    <div class="col">
      <h3>Format Text</h3>
      <p>Headers</p>
      <pre>
# This is an &lt;h1&gt; tag
## This is an &lt;h2&gt; tag
###### This is an &lt;h6&gt; tag</pre>
     <p>Text styles</p>
     <pre>
*This text will be italic*
_This will also be italic_
**This text will be bold**
__This will also be bold__

*You **can** combine them*
</pre>
    </div>
    <div class="col">
      <h3>Lists</h3>
      <p>Unordered</p>
      <pre>
* Item 1
* Item 2
  * Item 2a
  * Item 2b</pre>
     <p>Ordered</p>
     <pre>
1. Item 1
2. Item 2
3. Item 3
   * Item 3a
   * Item 3b</pre>
    </div>

      <h3>Miscellaneous</h3>

Format: ![Alt Text](url)
</pre>

     <pre>

[GitHub](http://github.com)</pre>

As Kanye West said:
